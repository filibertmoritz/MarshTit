# Marsh Tit Distribution Model

We aim to understand the distribution and habitat requirements of **Marsh Tits** in Sweden, especially the southermost county Skane. Therefore, we employ observations obtained from the GBIF database (mostly originating from Artportalen) in a presence-only species distribution modelling approach. 

This project is part of the course BIOR83: 'Conservation Biology' at Lund University.
